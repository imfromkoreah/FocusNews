## 🔍📰: 포뉴(FocusNews)

**짧지만 핵심적인 뉴스가 필요해?"**

짧고 간결한 컨텐츠를 소비하는 사회 흐름에 맞춘 뉴스 요약 챗봇입니다!<br>
요약된 핵심 뉴스를 전달해 드릴게요!

### 🔔 핵심 기술 소개
#### 	1. 네이버 검색 API를 사용한 뉴스 데이터 크롤링

#### 2. OpenAI의 GPT API를 사용한 AI 기반 뉴스 요약 및 분석

#### 3. Karlo API를 이용한 미디어 콘텐츠(카드뉴스) 자동화
